# TensorBoard Plot Exports

Export plots from TensorBoard by:
1. Open TensorBoard: `tensorboard --logdir=zbot_walking_task/`
2. Navigate to the plot you want
3. Click the download icon or screenshot

## Required plots:
- `reward_total.png` - Total reward over training
- `reward_forward.png` - Forward velocity reward

## How to export from TensorBoard:
1. Hover over a chart
2. Click the "Download PNG" button (camera icon)
3. Save to this folder
